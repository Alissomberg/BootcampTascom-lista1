# Bootcamp Tascom & ETE JBL - Lista de Exercícios 01

Este repositório foi criado para documentar e organizar a resolução da lista 1 do Bootcamp Tascom realizado em parceria com a ETE JBL.

## Sobre o Projeto
O objetivo desta etapa foi consolidar os pilares fundamentais da programação, garantindo uma base sólida para o desenvolvimento de sistemas mais complexos. Os exercícios foram estruturados em 3 níveis para facilitar a progressão do aprendizado.

## O que foi aprendido
* Variáveis e operadores.
* Operadores aritméticos básicos.
* Exibição de dados no console.
* Versionamento de código utilizando Git e GitHub.

## Tecnologias utilizadas
* JavaScript
* Node.js
* Git e GitHub


## Como usar
Com o Node.js instalado, siga os passos abaixo:

1. Abra o terminal e clone o repostiório utilizando: 
```bash
git clone https://github.com/Alissomberg/BootcampTascom-lista1.git
``
2. Navegue até a pasta do nível desejado (ex: Nível 1)
3. Abra um terminal na pasta escolhida e então digite:
```bash 
node nome-do-arquivo.js
